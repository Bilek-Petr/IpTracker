## HOW TO USE THE APP

in case of downloading just use npm run dev and it's ready to go.

as an user you can try following IPs
- 142.197.151.12
- 98.53.56.103
- 48.117.7.197
- 102.197.197.197

or use random [IP generator](https://www.ipvoid.com/random-ip/)
